# Usados Europa

Landing page para **Usados Europa** — importación de maquinaria usada de Europa (agrícola, vial y de construcción) hacia Argentina.

## 🌐 URL

[https://www.usadoseuropa.com.ar](https://www.usadoseuropa.com.ar)

## Estructura

```
├── index.html    # Página principal
├── style.css     # Estilos (dark mode, responsive)
├── hero.png      # Imagen hero de maquinaria
├── favicon.png   # Favicon
└── README.md
```

## Stack

- **HTML5** semántico
- **CSS3** puro (sin frameworks)
- **Google Fonts** — [Inter](https://fonts.google.com/specimen/Inter)

## Características

- Diseño dark mode premium con paleta navy/gold
- Hero full-viewport con imagen de fondo y gradiente
- Navbar fijo con efecto glassmorphism (backdrop-filter)
- Cards informativas con hover animations
- Banner de "sitio en desarrollo" con indicador pulsante
- Sección de contacto (WhatsApp + Email)
- Totalmente responsive (mobile-first breakpoint a 640px)
